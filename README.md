# Snowflake Cortex Analytics Assistant

## Overview
AI-powered analytics assistant that converts natural language business
questions into Snowflake SQL using Snowflake Cortex LLMs.

## How to Run
1. Open `cortex_analytics.sql`
2. Execute queries from top to bottom in Snowflake

## Example Question
"What are the top 3 products by total sales amount?"

## Technologies
- Snowflake
- Snowflake Cortex LLM
- SQL
